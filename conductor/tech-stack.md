# Technology Stack: AgriBid

## Frontend
- **Framework**: [React](https://react.dev/) (powered by [Vite](https://vitejs.dev/))
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: [Convex React Hooks](https://docs.convex.dev/client/react)

## Backend & Database
- **Platform**: [Convex](https://www.convex.dev/) (Real-time backend, database, and file storage)
- **Authentication**: [Better Auth](https://www.better-auth.com/) (integrated with Convex)

## Tooling & Infrastructure
- **Build Tool**: Vite
- **Package Manager**: npm (standard for this project)
- **Linting & Formatting**: ESLint
- **Git Hooks**: Husky + lint-staged
